## CampusTalks
## Live Link -> https://campustalks-o15c.onrender.com/
---
CampusTalks appears to be a full-stack social / discussion platform for students where users can connect, share ideas, and interact within a campus community.
Think of it like a mini social network for college students 🎓

---
## ✨ Key Features

### 🔐 User Authentication
- Secure login/signup using JWT
- User sessions handled via tokens
- Protected routes for authenticated users

### 💬 Social Interaction
- Users can create posts and discussions
- Interact via comments and replies
- Like or react to content

### 🖼️ Media Uploads
- Image uploads supported via Cloudinary
- Users can attach images to posts and profiles

### 👤 User Profiles
- Profile creation and customization
- View other users' profiles

### 🌐 Real-Time / Dynamic UI
- Smooth frontend updates using React
- API-based communication with backend

### 🔒 Environment-Based Configuration
- Uses `.env` for:
  - Database credentials
  - API keys
  - JWT secrets
  - Cloudinary credentials

## 🛠️ Tech Stack & Technologies

### Frontend
- React
- Vite
- JavaScript (ES6+)
- CSS / Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
---

### Setup .env file

- `MONGODB_URI=your_mongodb_connection_string`
- `PORT=5001`
